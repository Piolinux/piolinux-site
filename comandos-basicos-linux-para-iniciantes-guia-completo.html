Comandos básicos Linux para iniciantes: Guia completo

Quer dominar o terminal Linux, tornar-se mais produtivo no Sistema? Aprenda os comandos básicos. Não existem fadas ou bruxas para fazer mágica no aprendizado — o primeiro passo é seu. Existem diversos tipos de terminais no Linux, o mais conhecido é o Bash. Aqui no terminal, você usará "feitiços" (comandos), e é importante saber que alguns deles podem até inutilizar o sistema.

Através deste guia de comandos essenciais, leia atentamente o propósito de cada um. E lembre-se: rm -f /home/ não é um bom exemplo para iniciantes.

O que você vai aprender

  • Comandos para navegar entre pastas
  • Como listar arquivos e verificar permissões
  • Comandos para manipular arquivos e diretórios
  • Como obter ajuda e informações do sistema

Navegação no sistema de arquivos

Para se mover entre pastas, os comandos mais usados são:

  • pwd – mostra o diretório atual
  • ls – lista os arquivos e pastas no diretório
  • cd nome-da-pasta – muda para a pasta especificada
  • cd .. – volta para a pasta anterior
Exemplo: cd Documentos leva você para a pasta Documentos.

Listando arquivos e detalhes importantes

O comando ls tem várias opções para mostrar informações diferentes:

  • ls -l: lista arquivos com detalhes como permissões, dono e tamanho
  • ls -a: lista todos os arquivos, incluindo ocultos

Manipulando arquivos e diretórios

Comandos para criar, copiar, mover ou apagar arquivos e pastas:

  • mkdir nome-da-pasta: cria uma nova pasta
  • touch nome-do-arquivo: cria um novo arquivo vazio
  • cp origem destino: copia arquivos ou pastas
  • mv origem destino: move ou renomeia arquivos e pastas
  • rm nome-do-arquivo: remove arquivos (cuidado!)
  • rm -r nome-da-pasta: remove pastas e todo conteúdo dentro

Obter ajuda e informações do sistema

Se tiver dúvidas sobre um comando, use:

  • man comando: mostra o manual do comando
  • comando --help: mostra ajuda rápida do comando

Para verificar informações do sistema:

  • uname -a: mostra dados do kernel
  • top: exibe processos em execução em tempo real

Conclusão

Dominar esses comandos básicos do Linux abrirá muitas portas para você no sistema. Com prática, o terminal deixará de ser um mistério e se tornará uma ferramenta útil no seu dia a dia.

Comece aos poucos, experimente cada comando e veja como seu conhecimento vai crescer rapidamente.
