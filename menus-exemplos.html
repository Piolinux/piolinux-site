3️⃣ Menu com Dropdown (Submenu)

Início
Serviços
Web Mobile Infra 
<div class="menu3">
    <a href="#">Início</a>
    <div class="dropdown">
        <a href="#">Serviços</a>
        <div class="dropdown-content">
            <a href="#">Web</a>
            <a href="#">Mobile</a>
            <a href="#">Infra</a>
        </div>
    </div>
    <a href="#">Contato</a>
</div>

4️⃣ Menu com Ícones (sem dependências externas)

    <a href="#">🏠 Início</a>
    <a href="#">📄 Artigos</a>
