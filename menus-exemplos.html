📂 Exemplos de Menus Horizontais em HTML

1️⃣ Menu Horizontal Simples

1 2 3 4

🧾 Código:

<div class="menu1">
    <a href="#">1</a>
    <a href="#">2</a>
    <a href="#">3</a>
    <a href="#">4</a>
</div>

2️⃣ Menu com Hover Animado

Casa Artigos Contato

🧾 Código:

<div class="menu2">
    <a href="#">Casa</a>
    <a href="#">Artigos</a>
    <a href="#">Contato</a>
</div>

3️⃣ Menu com Dropdown (Submenu)

Início
Serviços
Web Mobile Infra
Contato

🧾 Código:

<div class="menu3">
    <a href="#">Início</a>
    <div class="dropdown">
        <a href="#">Serviços</a>
        <div class="dropdown-content">
            <a href="#">Web</a>
            <a href="#">Mobile</a>
            <a href="#">Infra</a>
        </div>
    </div>
    <a href="#">Contato</a>
</div>

4️⃣ Menu com Ícones (sem dependências externas)

🏠 Início 📄 Artigos 📧 Contato

🧾 Código:

<div class="menu4">
    <a href="#">🏠 Início</a>
    <a href="#">📄 Artigos</a>
    <a href="#">📧 Contato</a>
</div>
