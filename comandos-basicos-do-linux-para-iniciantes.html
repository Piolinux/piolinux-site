Comandos básicos Linux para iniciantes

Navegação no sistema de arquivos

Para se mover entre pastas, os comandos mais usados são:

  • pwd – mostra o diretório atual.
  • ls – lista os arquivos e pastas no diretório.
  • cd nome-da-pasta – muda para a pasta especificada.
  • cd .. – volta para a pasta anterior.
Exemplo: cd Documentos leva você para a pasta Documentos.

Listando arquivos e detalhes importantes

O comando ls tem várias opções para mostrar informações diferentes:

  • ls -l – lista arquivos com detalhes como permissões, dono e tamanho.
  • ls -a – lista todos os arquivos, incluindo ocultos.

Manipulando arquivos e diretórios

Use estes comandos para criar, copiar, mover ou apagar arquivos e pastas:

  • mkdir nome-da-pasta – cria uma nova pasta.
  • touch nome-do-arquivo – cria um novo arquivo vazio.
  • cp origem destino – copia arquivos ou pastas.
  • mv origem destino – move ou renomeia arquivos e pastas.
  • rm nome-do-arquivo – remove arquivos (atenção ao uso).
  • rm -r nome-da-pasta – remove pastas e todo conteúdo dentro delas.

Obter ajuda e informações do sistema

Se tiver dúvidas sobre um comando, utilize:

  • man comando – mostra o manual completo do comando.
  • comando --help – exibe ajuda rápida com as opções do comando.

Para verificar informações do sistema:

  • uname -a – mostra dados completos sobre o kernel.
  • top – exibe os processos ativos em tempo real.

FAQ

Como navegar entre pastas no Linux?

Use o comando cd para mudar de diretório.

Posso apagar arquivos pelo terminal?

Sim, com o comando rm, mas tenha cuidado para não apagar arquivos importantes.

Como ver detalhes dos arquivos?

Use ls -l para listar arquivos com informações detalhadas.
