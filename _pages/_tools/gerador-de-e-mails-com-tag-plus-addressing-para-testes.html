Uso. Formatos. Compatibilidades. Observações Técnicas.
Rastreamento de origem usuario+newsletter@dominio.com Gmail, Proton, Fastmail, Zoho, Mailfence. Funciona onde o MTA respeita o padrão RFC 5233 (subaddressing)
Filtragem automática Regra no cliente: redirecionar +loja para pasta "Compras" Thunderbird, Mutt, NeoMutt, Apple Mail, Outlook (com regras). Configure filtro por "Para contém +loja"
Testes de desenvolvimento dev+feature-x@seudominio.com Qualquer sistema com MTA compatível (Postfix, Dovecot, etc.). Útil para isolar logs de envio por funcionalidade.
Serviços que NÃO aceitam `+` — Microsoft 365 (alguns tenants), Outlook.com (antigo), alguns SaaS legados. Teste antes de usar em produção.
