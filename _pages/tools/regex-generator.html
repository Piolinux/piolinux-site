# Valida email técnico
• ^ : início da string
• [a-zA-Z0-9._%+-]+ : caracteres antes do @
• @ : caractere literal
• [a-zA-Z0-9.-]+ : domínio
• \.[a-zA-Z]{2,} : TLD (.com, .org, etc)
• $ : fim da string

Pra que serve Regex, ou expressão regular?

É tipo um código que encontra um padrão em textos, utilizada na ciência da computação?A gente usa pra checar se algo tá certo, pegar informações específicas ou trocar um pedaço do texto por outro.

Como utilizar esta ferramenta?

Escolha um padrão comum (email, URL, cep, telefone), copie a regex gerada e use em seu código, detalhe é gerado uma explicação.
