O que é um Gradient CSS?

É uma transição suave entre duas ou mais cores, usada como background em elementos web — adiciona profundidade e modernidade ao design, para seu site fica bonito.

Como usar esta ferramenta?

Escolha o tipo linear (vertical) ou radial (horizontal), ajuste as cores e o ângulo (se aplicável), faça testes.
